ООО ПК "Теплый Стан" является официальным дилером продукции PAROC по Уральскому и Сибирскому Федеральным округам

ЗАО "ПАРОК" ( PAROC Финляндия ) занимается производством технической теплоизоляцией для промышленности, строительства жилых домов, судостроительной индустрии, а также для предприятий, занимающихся производством машин и оборудования по следующим направлениям:

  • теплоизоляция для технологических трубопроводов, трубопроводов тепловых станций, систем отопления и водопроводов горячей воды в зданиях
  • теплоизоляция водопроводов холодной воды
  • пластмассовые покрытия из ПВХ для теплоизоляции труб
  • теплоизоляция технологических и складских резервуаров
  • теплоизоляция, звукоизоляция и противопожарная изоляция для вентиляционных каналов, огнезащита воздуховодов
  • теплоизоляция и противопожарная изоляция (огнезащита) для дымовых каналов и выхлопных труб
  • теплоизоляция для котлов, печей, теплообменников и теплотехнического оборудования
  • теплоизоляция и противопожарная изоляция (огнезащита) для строительных деталей заводского изготовления, различных машин и оборудования

Теплоизоляционные материалы марки PAROC / ПАРОК изготавливаются из каменной ваты на основе базальтового волокна. Большинство стандартных изделий имеют максимальную рабочую температуру 7500С, а высокотемпературные изделия 9000С. У каменной ваты ( БАЗАЛЬТОВАЯ ТЕПЛОИЗОЛЯЦИЯ ) PAROC / ПАРОК температура спекания составляет 10000С.

Каменная вата обладает высокой механической прочностью и химической стойкостью, не горит и обладает выраженными водоотталкивающими свойствами. Каменная вата обладает высокими теплоизоляционными свойствами в пределах широкого температурного диапазона.

Увеличивая рентабельность и повышая комфорт

Отопление, вентиляция и системы кондиционирования воздуха создают необходимые условия для комфортного и безопасного проживания и работы. Для того, чтобы эти системы работали с максимальной эффективностью, они должны быть изолированы с помощью соответствующих элементов технической изоляции.
